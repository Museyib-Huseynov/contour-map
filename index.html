Rules for uploading Excel file (.xlsx):

  • Only the first sheet will be parsed;
  • 4 columns required;
  • First row is for headers: wellNames, x, y, z (order does not matter);
  • x (East), y (North) are coordinates;
  • z might be pressure, water cut, TVD etc.
